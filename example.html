Practical Solved Examples for PHP Concepts

Review these simple, solved examples to understand how basic PHP concepts are implemented.

1. Input Data: Storing User Input

Example: Processing a Form Submission

We simulate receiving data (Name, Age) from an HTML form using the global $_POST array.


<?php
    // هذا يحاكي البيانات القادمة من الحقل (input field)
    $username = $_POST['user_name'] ?? 'Guest'; 
    $user_age = $_POST['user_age'] ?? 30;
    
    echo "Hello, " . $username . "! Your calculated age is " . $user_age . ".";
?>

Output: (If user inputs "Sara" and "28")

Hello, Sara! Your calculated age is 28.

2. Process: Basic Numeric Operators

Example: Calculating Simple Interest

We use multiplication and division operators to calculate the simple interest on a loan.


<?php
    $principal = 1000;
    $rate = 0.05; // 5% annual rate
    $time = 3;    // 3 years

    // Formula: Interest = Principal * Rate * Time
    $interest = $principal * $rate * $time;

    echo "Total Simple Interest: $" . $interest;
?>

Output:

Total Simple Interest: $150

3. Process: Conditions (IF, ELSEIF, ELSE)

Example: Grading System

We use conditional statements to assign a letter grade based on a numerical score.


<?php
    $score = 85;
    $grade = "";

    if ($score >= 90) {
        $grade = "A";
    } elseif ($score >= 80) {
        $grade = "B";
    } else {
        $grade = "C or lower";
    }

    echo "The student's grade is: " . $grade;
?>

Output: (If score is 85)

The student's grade is: B

4. Process: Loops (FOR Loop)

Example: Multiplication Table

Using a **`for` loop** to generate the multiplication table for the number 7 from 1 to 5.


<?php
    $number = 7;
    for ($i = 1; $i <= 5; $i++) {
        $result = $number * $i;
        echo $number . " x " . $i . " = " . $result . "<br>";
    }
?>

Output:

7 x 1 = 7

7 x 2 = 14

7 x 3 = 21

7 x 4 = 28

7 x 5 = 35

5. Output Data: Displaying Output

Example: Combining PHP and HTML Output

Demonstrates using **`echo`** to combine strings, variables, and HTML tags for formatted output.


<?php
    $message = "Welcome to PHP Zone";
    $version = "8.3";
    
    // استخدام علامة H3 وربط المتغيرات
    echo "<h3 style='color: #4F5D95;'>" . $message . "</h3>";
    echo "<p>The current version is: " . $version . "</p>";
?>

Output:

Welcome to PHP Zone

The current version is: 8.3
